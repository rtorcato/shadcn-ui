# shadcn-ui

`@rtorcato/shadcn-ui` is a React component library built on shadcn/ui, Radix UI,
TypeScript, and Tailwind CSS v4. It publishes to the public npm registry.

:::info Docs are under construction
This site is being built out. Tracking issue:
[#12](https://github.com/rtorcato/shadcn-ui/issues/12).

- Brand theme + landing page — [#7](https://github.com/rtorcato/shadcn-ui/issues/7)
- Component pages with live previews — [#8](https://github.com/rtorcato/shadcn-ui/issues/8)
- TypeDoc API reference — [#9](https://github.com/rtorcato/shadcn-ui/issues/9)
:::

## Install

```bash
pnpm add @rtorcato/shadcn-ui
```
